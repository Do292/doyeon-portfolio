Project Overview

다수의 제조 솔루션(Modeler, MES, SPC 등)을 하나의 UI 체계로 통합하는 프로젝트로, 제품화 기준에 맞춘 UI 정비 및 퍼블리싱 체계화를 핵심 목표로 함. Figma 기반 시안 제작부터 SCSS 구조 정리, 테마 분리 및 다크모드 적용까지 전반적인 구조 설계를 주도했으며, Vue 및 Syncfusion 컴포넌트 기반의 퍼블리싱도 함께 수행.

UI 표준화 Concept

Nexbe Suite 제품군 통합 과정에서 여러 솔루션(Modeler, MES, SPC 등)의 구분을 직관적으로 인식할 수 있도록, 각 제품마다 고유 테마 컬러를 지정하고 이를 SCSS 기반으로 분리해 적용. 상단 헤더의 제품 탭을 선택하면 전체 페이지의 주요 포인트 색상이 해당 제품의 컬러로 동적으로 변경되며, 이를 통해 사용자는 각 제품 간 전환을 자연스럽게 인식할 수 있도록 구성.

기능제안

  • 메뉴를 직관적으로 보고싶다는 니즈가많아 리본형식으로 구성
  • 메뉴를 아이콘이 크게보이는 리본뿐아니라 아이콘+텍스트, 텍스트, 다양한 조합으로 니즈를 만족시키려 노력했다
  • 스플릿터를 활용해서 한눈에 보고싶은 니즈 반영

업무내용

  • Figma 기반 UI 시안 제작 및 전체 퍼블리싱 구조 설계
  • 누적으로 복잡해진 CSS/SCSS 자산을 경량화하고 공통 스타일 체계로 재정리
  • 제품별 테마 컬러 기반 SCSS 분리 및 다크모드 적용으로 제품 간 시각 흐름 분리
  • 시각 요소의 일관성 확보 및 기능 유지 중심의 퍼블리싱 정비
  • Syncfusion UI(Grid, Chart 등)의 기본 구조 분석 및 디자인 시안 기반 스타일 커스터마이징 수행
  • Vue 컴포넌트 구조(slot, props 흐름 등)에 맞춘 마크업 구조에 따라 마크업 작업을 수행하고,일부 UI 동작 요소 구현에 참여
  • Syncfusion 컴포넌트를 활용한 API 연동 및 그리드 기능 구현에 협업 참여
