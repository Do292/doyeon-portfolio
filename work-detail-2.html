Project Overview

롯데 측 사내 시스템(MCS)의 맞춤형 UI를 기획 및 퍼블리싱한 파견 프로젝트. 고객 요구사항에 따라 UI 흐름을 재설계하고, SVG 기반 설비 UI 및 샘플 시안 제안, 스타일 정의 등을 수행. 시안 및 구조 설계와 함께 UI 제안서 작성까지 담당.

도면작업, 아이콘작업

  • illustrator로 공정 도면 제작 → 규칙적 id 부여 후 SVG로 내보냄.
  • 프런트에서 SVG DOM 파싱해 id ↔ 장비 매핑 구성.
  • 실시간 좌표 (x, y, θ)로 transform(translate/rotate) 적용해 장비 이동 구현.
  • Stocker/장비 상태는 .on/.off 클래스 토글로 적재·가동·알람 표시.
  • 결과: 해상도 독립, 도면 교체만으로 리비전 가능, 의존성↓/성능↑, 운영 가독성 향상.

기능리스트 작성

내가 맡은 대시보드·설비 뷰 영역을 중심으로 화면 구조와 표시 규칙을 문서화.
Illustrator→SVG 파이프라인(네이밍·좌표계)과 상태 표기(on/off, alarm), 반응형·스크롤 기준, 로딩/빈 상태 UI 등을 정리해 협업 시 참고 자료로 제공.

업무내용

  • 고객 요구사항 기반 UI 흐름 설계 및 Figma 시안 제작, 개선 방향 기획
  • 고객사에 UI 개선 방향 제안서 작성 및 회의를 통해 반영 결정
  • 설계, 제안, 퍼블리싱까지 UI 기획 전반에 실무적으로 참여
  • 샘플 페이지 구성 및 전체 스타일 방향 제안
  • Vue 컴포넌트 구조(slot, props 흐름 등)에 맞춘 마크업 구조에 따라 마크업 작업을 수행하고,일부 UI 동작 요소 구현에 참여
  • 기능 요소에 대한 스타일 정의 및 유지보수 대응
