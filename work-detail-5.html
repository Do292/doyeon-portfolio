Overview

자사 제품의 브랜드 강화를 위한 홈페이지 및 e-브로슈어 전면 리뉴얼 프로젝트. 브랜드 네이밍부터 Figma 시안 제작, Wordpress 기반 반응형 퍼블리싱까지 수행하였습니다. 시각적 일관성과 사용자 흐름을 고려한 콘텐츠 구조 설계 중심의 퍼블리싱을 담당했습니다.

브랜딩

  • 자사 솔루션군을 4개 브랜드로 재편하고, 각 제품군의 네이밍과 로고 시그니처(워드마크)를 디자인했습니다.
    • nexbe+ : 자동화 운영·제어 통합 솔루션(MES, EAS, MCS, WMS, FMB 등)
    • boostbe+ : 제조 운영 분석 솔루션(YMS, FDC/EPT, RPT)
    • ezieco : 통합 개발·운영 지원 도구(eziframe, eziflow, ezicrtl, ezihub)
    • IOBridge : 장비 자동화 핵심 솔루션(BridgeX, BridgeP)
  • 30년의 신뢰와 기술 경쟁력을 동시에 전달하기 위해 라이트/다크 컬러웨이·가로/스택형 락업을 제작하고 사용 가이드를 정리했습니다.

업무내용

  • Figma 기반 UI 시안 제작 및 전체 퍼블리싱 구조 설계
  • 누적으로 복잡해진 CSS/SCSS 자산을 경량화하고 공통 스타일 체계로 재정리
  • 제품별 테마 컬러 기반 SCSS 분리 및 다크모드 적용으로 제품 간 시각 흐름 분리
  • 시각 요소의 일관성 확보 및 기능 유지 중심의 퍼블리싱 정비
  • Syncfusion UI(Grid, Chart 등)의 기본 구조 분석 및 디자인 시안 기반 스타일 커스터마이징 수행
  • Vue 컴포넌트 구조(slot, props 흐름 등)에 맞춘 마크업 구조에 따라 마크업 작업을 수행하고,일부 UI 동작 요소 구현에 참여
  • Syncfusion 컴포넌트를 활용한 API 연동 및 그리드 기능 구현에 협업 참여
